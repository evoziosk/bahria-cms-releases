# Bahria CMS — App Releases

This repository hosts the official APK releases for the **Bahria University CMS/LMS** Android app.

## Download

Get the latest APK from the [**Releases**](../../releases/latest) page.

> **Note:** You may need to enable "Install from unknown sources" in your Android settings before installing.

## About

The Bahria CMS app provides students with a convenient way to access:
- Assignment submissions and grades
- Attendance records
- Lecture notes and materials
- Exam seating plans and admit slips
- Course announcements
- AI-powered summaries

## Source Code

The source code is maintained in a private repository. Releases are published here automatically via CI/CD on every push to `main`.

## Install Instructions

1. Download `bahria-cms.apk` from the latest release
2. Open the file on your Android device
3. Allow installation from unknown sources if prompted
4. Log in with your Bahria University CMS credentials
